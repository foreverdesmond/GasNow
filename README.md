# GasNow

GasNow is a comprehensive solution for managing gas fee data and interactions with blockchain networks. This project includes a class library, an API, a console application, a test suite, and a frontend web application.

## Table of Contents

- [Features](#features)
- [Components](#components)
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)
- [License](#license)

## Features

- Real-time gas fee information retrieval.
- API endpoints for accessing gas fee data.
- Console application for quick interactions with the GasNow library.
- Unit tests to ensure code quality and reliability.
- Frontend web application for user-friendly access to gas fee data.

## Components

1. **GasNow Class Library**: 
   - A library for managing gas fee data and interactions with blockchain networks.

2. **GasNow.API**: 
   - An API project that provides endpoints for accessing gas fee information.

3. **GasNowConsole**: 
   - A console application for interacting with the GasNow library and performing operations.

4. **GasNowTest**: 
   - A test project that includes unit tests for the GasNow library and API.

5. **gas-now-web**: 
   - A frontend application for displaying gas fee data and interacting with the API.

## Installation

To get started with the GasNow project, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/foreverdesmond/GasNow.git
   ```

2. Navigate to the project directory:

   ```bash
   cd GasNow
   ```

3. Restore the NuGet packages for the .NET projects:

   ```bash
   dotnet restore
   ```

4. For the frontend application, navigate to the `gas-now-web` directory and install the dependencies:

   ```bash
   cd gas-now-web
   npm install
   ```

## Usage

### Using the Console Application

To use the console application, navigate to the `GasNowConsole` project directory and run:

  ```bash
  dotnet run
  ```

### Running the API

To run the API, navigate to the `GasNow.API` project directory and use the following command:

  ```bash
  dotnet run
  ```

### Accessing the Frontend

To run the frontend application, navigate to the `gas-now-web` directory and use:

  ```bash
  npm start
  ```

This will start the development server, and you can access the application in your web browser at `http://localhost:3000`.
